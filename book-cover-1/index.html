CSS book covers

This project was originally intended to be a simple book cover, but as I went on I decided, why not add a little 3D into the mix? So I turned my book a few degrees around a couple axes.

But then I realised that I could apply transitions to 3D transformations, so I did. Only... now the book looked so much like something out of a bookstore's e-commerce page that I simply couldn't leave it alone, so I put two other books beside it, changed their styles up a bit and put them all in a black box.

Play around with the devtools if you want! This page has zero javascript.
